What is React Query?

"The missing data-fetching library for web applications"

"It makes fetching, caching, synchronizing and updating

Challenges with Server Data

  • Caching... (possibly the hardest thing to do in programming)
  • Deduping multiple requests for the same data into a single request
  • Updating "out of date" data in the background
  • Knowing when data is "out of date"
  • Reflecting updates to data as quickly as possible
  • Performance optimizations like pagination and lazy loading data
  • Managing memory and garbage collection of server state
  • Memoizing query results with structural sharing

Dependent Queries

                
                    const { data: user } = useQuery({
                        queryKey: ['usersByEmail', email], 
                        queryFn: () => fetchUserByEmail(email)
                    });

                    const { data: posts } = useQuery({
                        queryKey: ['users', user?.id, 'posts'], 
                        queryFn: fetchPosts, 
                        enabled: !!user?.id,
                    });

Mutating Data

Mutating Data

                    
                        const addPostMutation = useMutation(
                            newPost => axios.post('/posts', newPost)
                        );

                        // sync
                        addPostMutation.mutate({ title: 'Ultimate TanStack Query Guide' });

                        // or async
                        await addPostMutation.mutateAsync({ title: 'Ultimate TanStack Query Guide' });

Query Invalidation

                    
                        const queryClient = useQueryClient();

                        // add/update posts
                        await addPostMutation.mutateAsync({ title });

                        // invalidate the posts query key to trigger a refetch (if needed)
                        await queryClient.invalidateQueries(['posts']);

                        // or, explictly refetch the posts query
                        await queryClient.refetchQueries({ queryKey: ['posts'] });

Automatic Refetching (polling)

                    
                        const { data } = useQuery({
                            queryKey: ['posts'],
                            queryFn: fetchPosts,
                            refetchInterval: 60000,
                        });
